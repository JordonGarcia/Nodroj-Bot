# To install WeatherBot with Node.js v14
  » In Terminal
   ### `git clone https://github.com/JordonGarcia/jordongarcia.com`
   ### `npm i yarn -S`



# To run the bot locally (Development Mode using Nodemon)
  » In Terminal
   ### `yarn dev`



# To run the bot locally (Production Mode)
  » In Terminal
   ### `yarn start`
